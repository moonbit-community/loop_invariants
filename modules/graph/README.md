# Graph Algorithms

This module contains graph and tree packages, including traversal, connectivity,
shortest paths, minimum spanning trees, network flow, matching, LCA, SCC, and
tree decomposition algorithms.

Packages are organized as one algorithm per directory. Each package contains a
`README.mbt.md` tutorial with examples that are checked by `moon test`.

Run from the workspace root:

```bash
moon check
moon test
moon info
moon fmt
```
