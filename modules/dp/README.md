# Dynamic Programming

This module contains dynamic-programming packages and optimizations, including
knapsack, LCS, LIS, edit distance, bitmask DP, matrix chain multiplication,
palindrome partitioning, and convex hull trick.

Packages are organized as one problem family or technique per directory. Each
package contains a `README.mbt.md` tutorial with examples that are checked by
`moon test`.

Run from the workspace root:

```bash
moon check
moon test
moon info
moon fmt
```
