# Persistent Data Structures

This module contains persistent arrays, stacks, queues, deques, Fenwick trees,
segment trees, heaps, tries, maps, sets, ropes, and balanced-tree examples.

Packages are organized as one persistent structure per directory. Each package
contains a `README.mbt.md` tutorial with examples that are checked by
`moon test`.

Run from the workspace root:

```bash
moon check
moon test
moon info
moon fmt
```
